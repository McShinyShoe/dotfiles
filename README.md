# My dotfiles :D
This repo contains the dotfiels that I am using for my devides. Feel free to take a look and copy it.

You should be able to link each folder to `~/.config` to use this setup, that way you can update it by pulling the repo.
``` sh
ln -s /this/repo/location/<component> ~/.config/<component>
```

> [!WARNING]
> Right now i am using `Cachy OS`, which is an arch based linux. I havent tested this on other linux distributions, so you might need to do extra configuration if you want to use this.

|Component|App Used|
|-|-|
|Desktop|`hyprland`|
|Shell|`fish`|
|Launcher|`rofi`|
|Terminal|`kitty`|
|Nontification|`mako`|
|TopBar|`waybar`|
|TextEditor|`code`|
**yes i use vs-code, sue me :D*
